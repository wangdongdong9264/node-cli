# Dongdong-cli

[Englist](./README.md) | 简体中文

## 说明

基于node `10.16.3`

## 开始

```
// 标准开头
#!/usr/bin/env node
```

### 使用

```bash

dongdong -h

```

### 测试

```sh
npm link
```

#### 取消私有模块

```sh
npm unlink <CliName>
```
